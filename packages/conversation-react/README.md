# @maia/conversation

Multi-agent conversation UI and conversation modeling for Maia.

## Install

```bash
npm install @maia/conversation
```

## Usage

```tsx
import { ConversationPanel } from "@maia/conversation";

<ConversationPanel rows={rows} loading={false} />;
```

## What it includes

- conversation row types
- conversation-to-ACP helpers
- grouping and roster derivation
- conversation thread and panel UI
